On this page, we provide an overview of common software packages used in computational chemistry and materials science. Modern software tools provide an essential platform to apply quantum mechanical theories and run advanced simulations on complex systems. They are powered by efficient algorithms and high-performance computing techniques, enabling simulation and modeling on an unprecedented scale.

The software packages discussed here contain major features such as:

  • Computing electronic structures based on density functional theory and wavefunction-based methods with various level of theory. This allows us to gain insights into the electronic properties and chemical bonding of materials and molecules.

  • Geometry optimization to find stable structures and transition states. This is crucial for understanding reaction mechanisms, conformational changes, and phase transformations.

  • Molecular dynamic simulations at different timescales, from femtosecond electronic dynamics to microsecond conformational changes. These help us study structural fluctuations, phase transitions, and non-equilibrium processes.

  • Spectral simulations such as IR, UV-Vis, X-ray absorption, and NMR. The computed spectra can be directly compared with experiments for validation and material characterization.

  • Thermodynamics calculations including free energy, entropy, enthalpy, heat capacity, etc. These are important for studying the driving forces of chemical reactions and phase changes.

  • Various post-processing tools to analyze, visualize and discover meaningful results from the raw simulation data and outputs.

Quantum Chemistry & First Principles software

Gaussian

Introduction

量子化学软件Gaussian 是一款商业软件,用于计算分子结构、性质与反应性。它的开发目的是将复杂的量子力学理论应用于化学与材料科学的研究与应用中。

Gaussian软件具体可以通过计算来解决以下内容:

  • 分子结构与电子性质: Gaussian可以计算分子的各种电子结构与轨道,如计算分子轨道(MO)、价层轨道(VB)、核密度等,从而推导出分子的化学键强度、电离能、电子亲和力、极化率等电子性质与参数。这为理解分子的结构与性质提供了重要信息。
  • 构型优化与转变: Gaussian可以对分子进行构型优化计算,找到其最稳定的几何构型,也可以查找其潜在的过渡态与转型产物。这些对研究反应机理与分子动力学至关重要。
  • 动力学模拟: Gaussian可进行分子动力学模拟,追踪分子在一段时间内的运动变化,包括转动、振动与整体运动变化,可用于研究分子在不同温度与环境下的演化过程。
  • 反应性研究: Gaussian可用于研究各种反应的可能性和机理,包括化学反应的速率论与动力学,以及光化学反应等,这可以指导和推动相关反应在实验室的实现。
  • 光谱计算: Gaussian可用于计算分子的各种光谱数据,如红外光谱、紫外吸收光谱、顺磁共振光谱等,这为光谱学研究与分析提供重要理论基础 。
  • 生物大分子模拟: Gaussian的最新版本已经可以用于蛋白质与生物大分子的模拟计算与性质研究,这为生物化学与生物物理学的理论研究提供了强有力的工具。

Gaussian软件的开发初衷是将量子化学理论应用于化学研究的每一个方面,其强大的计算功能大大简化了理论化学与计算化学的复杂计算,使量子力学理论的应用成为日常研究的一部分,成就了计算化学时代的到来。它被广泛用于学术与工业界,已经成为化学与材料研究中必不可少的工具。

CP2K

Introduction

CP2K软件是一款开源的第一性原理量子化学软件包,基于密度泛函理论,用于模拟凝聚态、表面与接口、生物体系以及各种材料的电子结构与性能。

它的开发目的是:

  • 将密度泛函理论作为一种高精度的第一性原理方法应用于凝聚态与材料科学研究。传统的Hartree-Fock方法和半经验方法的精度较低,CP2K 的开发满足了对更高精度与预测性理论方法的需求。

  • 提供一个从原子尺度出发研究复杂系统电子结构与性质的强有力工具。这些系统的性质难以从实验获得全面与深入理解,计算成为理解其中微观机理的重要手段。

  • 将高性能计算应用于大规模与复杂系统的模拟,实现从电子、原子到微观尺度的多层次模拟,填补化学与材料科学研究中的空白领域。

CP2K软件主要功能:

  • 电子结构计算:基于DFT计算分子和固体的电子轨道与带结构图,得到其电子密度、化学键强度、能隙等电子结构参数。

  • 构型优化:通过计算得到体系的最低能量构型与过渡态,用于研究反应机理与路径。

  • 分子动力学模拟:可模拟不同温度下体系原子/分子的运动变化,研究其热力学性质与过程。

  • 自由能计算:用于研究相变、吸附等过程的驱动力与可能性。

  • 光谱模拟:可计算各种光谱数据,为光谱学理论研究提供支持。

CP2K软件以其高精度、高可扩展性与高性能计算功能,已经成为表面、界面与生物大分子研究的重要工具。它推动了凝聚态与材料科学的第一性原理计算与理论研究,是理解和设计新材料的重要途径。CP2K软件由于提供开源的代码,也广受学术界的欢迎,有助于培养计算化学人才与交流合作。

VASP

Introduction

VASP软件是一款商业的第一性原理量子化学软件包,基于密度泛函理论,用于研究凝聚态、表面与界面、生物大分子等的电子结构、性质与过程。

它的开发目的是:

  • 提供一个高精度与高可扩展性的第一性原理计算工具,用于研究各种材料的电子结构与性质。密度泛函理论作为其理论基础,相比以往的Hartree-Fock理论和半经验方法,提供更高的预测精度。

  • 实现对复杂体系的大规模并行计算,以原子级出发研究这些体系的电子与能量特征,这些特征难以从实验得到全面理解。计算模拟成为理解其微观机理的重要手段。

  • 结合高性能计算,使得对含数以万计电子与原子的超大系统进行 first-principles 模拟成为可能,填补了理论与计算研究的重要空白领域。

VASP的主要功能是:

  • 基于密度泛函理论的电子结构计算:可得到体系的电子密度、轨道结构、能带结构、强相关体系的电子相互作用等信息。这为理解体系的电子机理与性质提供重要理论依据。

  • 构型优化与过渡态搜索:可寻找体系的最低能量构型与过渡态,为研究反应过程与路径提供信息。

  • 分子动力学模拟:可追踪不同温度下原子/分子的运动,研究体系的热力学性质,相变过程等。

  • 相图计算:可通过计算获得体系的压力-温度或组成-温度相图。

  • 其他应用:VASP还可用于计算光谱数据、磁性质、机械性质等。

VASP软件以其高精度、高性能与高可扩展性,已成为凝聚态与材料科学研究的重要第一性原理计算工具。它不仅用于各类材料的电子结构与性能研究,也广泛应用于表面、能源材料与生物大分子等领域。VASP软件推动了理论与计算材料科学的发展,是理解与发现新材料的重要途径。

ORCA

Introduction

ORCA软件是一款开源的第一性原理量子化学软件包,基于密度泛函理论,用于研究分子、聚合物与固体的电子结构与性质。

它的开发目的是:

  • 提供一个开源的DFT软件包,实现高精度的第一性原理计算,满足化学与材料科学对计算模拟工具的需求。开源代码有利于理论方法的广泛应用与发展。
  • 实现对大规模与复杂系统的并行计算,研究这些系统的电子结构与动态过程,这些研究难以仅靠实验完成。计算成为理解复杂体系性质的重要手段。
  • ORCA软件以高效的并行化算法与高性能计算功能,可研究包含数千个原子/电子的超大体系,弥补其他软件在这方面的不足,推动相关领域的发展。

ORCA软件的主要功能是:

  • 基于DFT的电子结构计算:可计算分子/固体的电子密度、轨道ENERGY、键长、热力学函数等,得到其电子结构特征。这为理解其性质与反应性提供重要的理论基础。
  • 构型优化与过渡态搜索:可获得体系的最稳定构型与过渡态几何结构,用于研究反应路径与过程。
  • 振动频率分析:可计算体系各维的振动模式与频率,研究其红外与拉曼光谱,以及热力学函数。
  • 时间相关DFT模拟:可用于研究体系在外场作用下(如激光场)的动态响应与电子重排过程。
  • 其他应用:ORCA也可用于研究体系的磁性质、光电子光谱、自由能、溶剂作用等。

ORCA软件以其高性能计算功能、开源代码和不断更新的理论方法,已广泛应用于有机分子、金属配合物、二维材料与生物大分子等各领域的研究。它推动了化学与材料科学领域第一性原理计算与理论研究工作,是设计与发现新材料的重要工具。

Python

Introduction

Python 是一门通用目的的编程语言,在科学计算、机器学习与软件开发等领域有着广泛的应用。

Python 的开发目的是:

  • 提供一门易读、简洁而强大的编程语言。Python 的语法简单明了,容易上手,但同时具有强大的功能,能胜任复杂的程序开发任务。这使其既适用于初学者,也满足专业程序员的需求。
  • 具有丰富的库和工具包,用于科学计算、数据分析、图像处理、机器学习等。这些工具大大简化了相关领域的程序开发过程,令 Python 成为这些领域最流行的编程语言之一。
  • 具有跨平台、可扩展和面向对象等特性,可应用于桌面终端、web 开发以及更复杂的软件构建。这使 Python 不仅适用于快速程序原型搭建,也能满足大项目的开发需求。

Python 主要应用于:

  • 科学计算与数据分析:有强大的 Numpy、Scipy、Matplotlib 等库,用于代数、信号与图像处理、统计建模、绘图等。

  • 机器学习:有TensorFlow、PyTorch、Scikit-learn 等机器学习与深度学习库。

  • 软件开发:可用于 web 应用、桌面应用以及大型程序框架的开发。

  • 系统管理与脚本:可用来编写各类管理工具与自动化脚本。

Python 的成功在于语言本身的优雅与清晰,以及丰富的第三方库。通过这些库,Python 覆盖了各个领域的主要应用,具有很强的表达能力和开发效率。

Python 一直保持着“简洁优先”(简洁就是美)的设计理念。它的简单易读与强大功能之间达到了极好的平衡,这也是它能适用于各种开发需求的原因。Python 以其友好、直观的语法和强大的功能被广泛应用于科研、工程与软件开发等领域。
