## 🐛 常见问题及解决

### 问题1: 登录后无法进入管理后台

**解决**：
1. 检查Token是否正确
2. 检查Token是否过期
3. 检查Token是否有repo权限
4. 清除localStorage后重试

### 问题2: "修改信息"页面不显示

**解决**：
1. 检查浏览器控制台是否有错误
2. 确认profile-editor.js已正确加载
3. 检查DOMContentLoaded事件是否触发

### 问题3: 表单没有默认值

**解决**：
1. 检查localStorage中是否有profile_data_v2
2. 检查ProfileEditor.loadCurrentData()是否正确执行
3. 检查populateForm()是否被调用

### 问题4: 点击"保存"没有反应

**解决**：
1. 检查浏览器控制台是否有错误
2. 确认所有必填字段都已填写
3. 检查网络连接是否正常
4. 检查GitHub Token是否有效

### 问题5: 主页没有更新

**解决**：
1. 等待1-2分钟（GitHub Pages部署时间）
2. 清除浏览器缓存
3. 使用无痕模式访问
4. 检查GitHub仓库是否已更新index.html
