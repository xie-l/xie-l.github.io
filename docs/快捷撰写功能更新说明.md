# 快捷撰写功能更新说明

**更新日期**: 2026-04-07
**版本**: v2.1.0

---

## 🎉 新功能

### **1. 新增两个快速记录分类**

在主页左下角点击铅笔图标（快捷撰写），现在可以选择：

- **❝ 摘录记录**: 收藏网上看到的精彩内容
- **💡 随笔思考**: 记录灵感闪现的瞬间（标题可选）

### **2. 标题可选功能**

**核心改进**: 当选择「随笔思考」分类时，标题不再是必填项！

**工作原理**:
- 如果填写标题 → 使用填写的标题
- 如果留空 → 自动提取内容前20字作为显示标题
- 文件名智能生成：有标题用标题，无标题用日期

---

## 🚀 使用方法

### **发布摘录记录**

1. 点击主页左下角 **铅笔图标** ✏️
2. 选择分类 **「摘录记录」**
3. 填写：
   - 标题（必填）
   - 来源链接（可选，建议填写）
   - 摘录内容（必填）
   - 标签（可选）
4. 点击「发布到博客」

### **发布随笔思考（无标题）**

1. 点击主页左下角 **铅笔图标** ✏️
2. 选择分类 **「随笔思考（标题可选）」**
3. 填写：
   - 标题**留空**
   - 思考内容（必填，至少20字）
   - 标签（可选）
4. 点击「发布到博客」

**示例**: 只写一句话「突然意识到氢能产业化的关键是商业模式」，标题留空，系统会自动提取前20字作为标题。

## 💡 使用技巧

### **何时使用「随笔思考（标题可选）」？**

✅ **适合场景**:
- 灵感闪现，来不及想标题
- 只想记录一句话或简短想法
- 快速记录，不追求完整文章
- 日常思考积累，后续可能扩展

❌ **不适合场景**:
- 需要清晰标识的长文章
- 系列文章或教程
- 对外分享的正式内容

### **标题自动提取规则**

```
内容: "突然意识到氢能产业化的关键是商业模式，而不是技术突破..."

自动标题: "突然意识到氢能产业化的关键..."
```

**规则**:
- 提取前20个字符
- 移除换行符
- 末尾添加省略号
