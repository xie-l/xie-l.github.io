## 🔍 故障排查

### 如果博客分类不显示

1. 检查是否已推送到GitHub
   ```bash
   git log --oneline -3
   ```

2. 检查blog/index.html文件
   ```bash
   grep -A 5 "blog-categories" blog/index.html
   ```

3. 清除浏览器缓存后重试

### 如果管理后台点击无反应

1. 打开浏览器开发者工具（F12）
2. 查看Console是否有JavaScript错误
3. 检查Network标签是否有失败的请求
4. 确认profile-editor.js已正确加载

### 如果文件上传失败

1. 检查文件大小是否超过10MB
2. 检查网络连接是否正常
3. 查看浏览器Console错误信息
4. 确认GitHub Token有写入权限

### 如果个人信息更新失败

1. 检查所有必填字段是否已填写
2. 查看浏览器Console错误信息
3. 确认GitHub Token有repo权限
4. 检查index.html是否在GitHub上已更新
