# 文字颜色优化方案

**日期**：2026年4月8日
**问题**："氢能驱动未来"文字与背景对比度不足
**解决方案**：修改文字颜色为深蓝色系渐变

---

## 一、问题分析

### 现状
- 背景：青色粒子（#00ffff）动态旋转
- 文字：白色到浅灰渐变（#ffffff → #e2e8f0 → #93c5fd）
- 问题：白色文字与白色/青色背景对比度不足，阅读费力

### 影响
- 用户体验差
- 可访问性（Accessibility）问题
- 不符合WCAG对比度标准

---

## 二、优化方案

### 方案：深蓝色系渐变

**修改位置**：`css/style.css` 第359-368行

**修改前**：
```css
.banner-title {
    font-size: 66px;
    font-weight: 800;
    margin-bottom: 18px;
    letter-spacing: -1.5px;
    line-height: 1.1;
    background: linear-gradient(135deg, #ffffff 0%, #e2e8f0 50%, #93c5fd 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
}
```

**修改后**：
```css
.banner-title {
    font-size: 66px;
    font-weight: 800;
    margin-bottom: 18px;
    letter-spacing: -1.5px;
    line-height: 1.1;
    background: linear-gradient(135deg, #0a1628 0%, #1e3a8a 50%, #06b6d4 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
    text-shadow: 0 2px 10px rgba(0, 0, 0, 0.3);
}
```

### 颜色选择理由

1. **#0a1628** (深蓝)：深色背景，与青色粒子形成强烈对比
2. **#1e3a8a** (中蓝)：中间色调，保持视觉层次
3. **#06b6d4** (青色)：与背景粒子同色但更深，保持主题一致性

4. **text-shadow**：添加阴影增强可读性，创造层次感

### 对比度分析

| 元素 | 颜色 | 对比度 | 可读性 |
|------|------|--------|--------|
| 背景粒子 | #00ffff | - | - |
| 文字起始 | #0a1628 | 高 | ✅ 优秀 |
| 文字中间 | #1e3a8a | 中 | ✅ 良好 |
| 文字结尾 | #06b6d4 | 中 | ✅ 良好 |

## 六、备选方案

### 方案B：添加文字背景遮罩
```css
.banner-content::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background: rgba(10, 22, 40, 0.3);
    z-index: -1;
}
```

**优点**：
- 保证文字可读性
- 不影响背景效果

**缺点**：
- 背景效果被遮挡
- 视觉层次减少

### 方案C：仅添加文字阴影
```css
.banner-title {
    text-shadow: 0 4px 20px rgba(0, 0, 0, 0.8);
}
```

**优点**：
- 改动最小
- 保持原有颜色

**缺点**：
- 对比度提升有限
- 阴影可能过重
