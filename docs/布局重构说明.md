# 布局重构说明

## 🎨 重大更新：全新上下布局

本次更新将个人主页从**左右结构**重构为**上下结构**，带来更现代、更流畅的浏览体验。

---

## 📋 主要改动

### 1. 布局结构变化

**旧布局（左右结构）**：
```
┌─────────────────────────────────────┐
│            导航栏                    │
├──────────┬──────────────────────────┤
│          │                          │
│  左侧    │                          │
│  头像    │     右侧内容区域          │
│  卡片    │                          │
│  固定    │                          │
│          │                          │
└──────────┴──────────────────────────┘
```

**新布局（上下结构）**：
```
┌─────────────────────────────────────┐
│            导航栏                    │
├─────────────────────────────────────┤
│                                     │
│      顶部个人信息区域（头像）        │
│                                     │
├─────────────────────────────────────┤
│                                     │
│      下方内容区域（可滚动）          │
│                                     │
└─────────────────────────────────────┘
```

### 2. 头像位置优化

- ✅ **头像放在页面顶部**，更加醒目
- ✅ **随页面滚动**，不再固定，体验更自然
- ✅ **居中显示**，完美解决遮挡问题
- ✅ **响应式设计**，移动端自动调整大小

### 3. 导航菜单优化

#### 区分链接类型

**单页锚点链接**（在当前页面跳转）：
- 首页
- 关于
- 教育
- 项目
- 技能
- 联系

**外部页面链接**（跳转到新页面）：
- 日记 📝
- 文件 📁

#### 视觉区分

- **分隔线**：在单页链接和外部链接之间添加垂直分隔线
- **外部链接标识**：外部链接右侧显示小箭头图标 ↗️
- **悬停效果**：鼠标悬停时箭头会轻微移动，提示可点击

## 🚀 如何使用

### 查看新布局

1. **推送到GitHub**：
   ```bash
   git push origin main
   ```

2. **清除浏览器缓存**：
   - Chrome/Edge: `Ctrl + Shift + R`
   - Firefox: `Ctrl + F5`
   - Safari: `Cmd + Option + E` 然后 `Cmd + R`

3. **访问网站**：
   - 主站：`https://xie-l.github.io/`
   - 日记：`https://xie-l.github.io/blog/`
   - 文件：`https://xie-l.github.io/files/`

### 调整头像位置

如果头像位置需要微调，编辑 `css/style.css`：

```css
.avatar-container {
    width: 150px;
    height: 150px;
    margin: 25px auto 20px;  /* 调整这个值 */
}
```

如果图片本身构图有问题，调整 `object-position`：

```css
.avatar {
    object-position: center 30%;  /* 调整这个值 */
}
```

### 自定义导航菜单

编辑 `index.html` 中的 `.nav-menu` 部分：

```html
<!-- 单页锚点链接 -->
<li class="nav-item">
    <a href="#section-id" class="nav-link">菜单名称</a>
</li>

<!-- 外部页面链接 -->
<li class="nav-item">
    <a href="folder/" class="nav-link nav-external">
        菜单名称
        <i class="fas fa-external-link-alt"></i>
    </a>
</li>
```

## 🔧 技术实现

### HTML结构

```html
<!-- 导航栏 -->
<nav class="navbar">
    <!-- 菜单项 -->
    <li class="nav-item">
        <a href="#about" class="nav-link">关于</a>
    </li>
    
    <!-- 分隔线 -->
    <li class="nav-divider"></li>
    
    <!-- 外部链接 -->
    <li class="nav-item">
        <a href="blog/" class="nav-link nav-external">
            日记
            <i class="fas fa-external-link-alt"></i>
        </a>
    </li>
</nav>

<!-- 顶部个人信息 -->
<section id="home" class="profile-header">
    <div class="profile-container">
        <div class="avatar-container">
            <img src="img/avatar.jpg" alt="个人头像" class="avatar">
        </div>
        <div class="profile-info">
            <h1 class="profile-name">谢亮</h1>
            <p class="profile-title">软件工程师</p>
            <!-- 其他信息 -->
        </div>
    </div>
</section>

<!-- 内容区域 -->
<main class="content-area">
    <!-- 各个section -->
</main>
```

### CSS关键样式

```css
/* 顶部个人信息 */
.profile-header {
    background: var(--card-bg);
    border-radius: 0 0 20px 20px;
    padding: 40px 20px;
    box-shadow: var(--shadow);
    margin-top: 70px;  /* 导航栏高度 */
    margin-bottom: 40px;
}

.profile-container {
    display: flex;
    align-items: center;
    gap: 40px;
    flex-wrap: wrap;
    justify-content: center;
}

/* 导航菜单分隔线 */
.nav-divider {
    width: 1px;
    height: 20px;
    background: var(--border-color);
    margin: 0 5px;
}

/* 外部链接标识 */
.nav-external i {
    font-size: 12px;
    opacity: 0.7;
    transition: var(--transition);
}

.nav-external:hover i {
    opacity: 1;
    transform: translateX(2px);
}
```

### JavaScript动画

```javascript
// 顶部个人信息区域淡入动画
setTimeout(() => {
    const profileHeader = document.querySelector('.profile-header');
    if (profileHeader) {
        profileHeader.style.opacity = '0';
        profileHeader.style.transform = 'translateY(-30px)'';
        
        setTimeout(() => {
            profileHeader.style.opacity = '1';
            profileHeader.style.transform = 'translateY(0)';
        }, 300);
    }
}, 100);
```
