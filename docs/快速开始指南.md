## 📋 功能使用说明

### 1. 头像居中显示 ✨

**问题已解决**：头像现在会自动居中显示，不会被遮挡。

如果还需要微调，编辑 `css/style.css`：

```css
.avatar-container {
    margin: 10px auto 20px;  /* 调整这个值 */
}
```

### 2. 浏览器图标美化 🎨

**已完成**：浏览器标签页现在显示个性化的👤图标。

如需更换图标，编辑 `index.html`：

```html
<link rel="icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>👤</text></svg>">
```

### 3. 个人日记系统 📝

#### 快速开始：

1. **访问日记页面**：点击导航栏的"日记"链接
2. **选择分类**：
   - **技术思考**：`blog/tech/`
   - **生活日记**：`blog/life/`

3. **添加新日记**：
   ```bash
   # 复制模板
   cp blog/tech/index.html blog/tech/2026-04-03-my-diary.html
   
   # 编辑内容
   nano blog/tech/2026-04-03-my-diary.html
   
   # 提交到Git
   git add blog/tech/2026-04-03-my-diary.html
   git commit -m "添加新日记"
   git push origin main
   ```

#### 详细指南：
查看 `blog/如何使用日记系统.md` 获取完整教程。

### 4. 文件管理中心 📁

#### 上传照片：

```bash
# 1. 复制照片到目录
cp ~/Pictures/my-photo.jpg files/photos/

# 2. 提交到Git
git add files/photos/my-photo.jpg
git commit -m "添加新照片"
git push origin main

# 3. 访问查看
# 打开 https://xie-l.github.io/files/photos/
```

#### 上传文档：

```bash
# 1. 复制文档到目录
cp ~/Documents/my-doc.pdf files/documents/

# 2. 提交到Git
git add files/documents/my-doc.pdf
git commit -m "添加新文档"
git push origin main

# 3. 访问查看
# 打开 https://xie-l.github.io/files/documents/
```

### 5. 丝滑滚动效果 ✨

**自动生效**：左侧个人资料卡片现在滚动更加流畅自然。

如需调整滚动效果，编辑 `js/main.js`：

```javascript
// 调整滚动敏感度
profileCard.style.transform = `translateY(${Math.min(maxOffset, 20)}px)`;
```

## 🆘 常见问题

### Q: 修改后网站没有更新？

A: GitHub Pages需要几分钟部署时间，请耐心等待。可以清除浏览器缓存后刷新。

### Q: 头像还是不居中？

A: 检查 `css/style.css` 中的 `.avatar-container` 样式，调整 `margin` 值。

### Q: 如何添加新导航菜单？

A: 编辑 `index.html`，在 `.nav-menu` 中添加新的 `<li class="nav-item">`。

### Q: 日记页面样式错乱？

A: 检查HTML文件路径是否正确，确保引用了正确的CSS文件。

### Q: 上传的文件无法访问？

A: 确认文件已提交到Git并推送到GitHub，检查文件路径是否正确。
