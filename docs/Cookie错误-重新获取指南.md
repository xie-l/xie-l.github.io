## 重新获取步骤

### 详细操作（带截图说明）

1. **打开浏览器**，访问 https://weread.qq.com

2. **扫码登录**微信读书
   - 确保登录成功
   - 能看到书架内容

3. **按F12打开开发者工具**
   - Mac: `Cmd + Option + I`
   - Windows: `F12` 或 `Ctrl + Shift + I`

4. **切换到Application标签**
   - 在开发者工具顶部菜单栏
   - 如果看不到，点击`>>`展开更多选项

5. **左侧选择Cookies → weread.qq.com**
   - 展开Cookies菜单
   - 点击weread.qq.com

6. **找到wr_skey这个Cookie**
   - 在Cookie列表中查找Name为`wr_skey`的行
   - 通常在最下面或中间位置

7. **完整复制Value列的值**
   - **关键**：必须完整复制整个Value！
   - 点击Value列的值（应该是一长串字符）
   - 右键 → Copy → Copy value
   - 或者双击选中，按Cmd+C复制

### 如何确认复制完整

**正确特征**：
- ✅ Value值很长（通常100+字符）
- ✅ 包含字母、数字、特殊符号
- ✅ 看起来像随机字符串

**错误特征**：
- ❌ 只有几个字符（如O1pp18ZR）
- ❌ 看起来像单词或缩写
- ❌ 明显太短

## 常见错误

### 错误1：只复制了部分值

**表现**：Cookie值很短（如O1pp18ZR）

**原因**：
- 双击时没有选中全部
- 手动选择时只选了一部分

**解决**：
- 右键 → Copy → Copy value（推荐）
- 或者三击Value列，然后复制

### 错误2：复制了错误的字段

**表现**：Cookie值不像随机字符串

**可能复制了**：
- Name列（wr_skey）
- Domain列（weread.qq.com）
- Path列（/）

**解决**：
- 确保复制的是Value列

### 错误3：Cookie被截断

**表现**：Value显示为省略号（...）

**原因**：
- 开发者工具界面显示不完整

**解决**：
- 双击Value列，进入编辑模式
- 全选并复制
- 或者右键 → Copy → Copy value
