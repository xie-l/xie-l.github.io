### 3.2 代码块

#### 行内代码

使用 `console.log('Hello World')` 输出日志。

#### 多行代码块

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome, ${name}`;
}

greet('Obsidian');
```

#### 带语法高亮的代码块

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
