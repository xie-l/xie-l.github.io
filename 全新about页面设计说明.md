# 全新 About 页面设计说明

## 🎨 设计理念

### 核心定位
**"氢能科研者 × 产业实践者"** - 突出学术背景与产业实践的双重身份

### 设计风格
- **现代科技风**：深蓝色背景 + 青绿色点缀，体现科技感
- **玻璃拟态（Glassmorphism）**：半透明卡片 + 模糊背景，营造层次感
- **暗黑模式优先**：保护视力，突出内容重点
- **微交互设计**：悬停动画、滚动动画、数字计数等细节

## ✨ 主要特性

### 1. 3D 粒子背景（Hero Section）
- **技术**：Three.js
- **效果**：5000个粒子组成的动态背景，随鼠标移动旋转
- **意义**：象征科研中的分子/原子运动，体现科技感

### 2. 滚动动画系统
- **技术**：Intersection Observer API
- **效果**：元素进入视口时淡入/滑入
- **类型**：
  - `fade-in`：淡入
  - `slide-in-left`：从左滑入
  - `slide-in-right`：从右滑入

### 3. 数据可视化
- **技术**：ECharts.js
- **内容**：论文发表趋势图
- **交互**：悬停显示详细信息

### 4. 技能条动画
- **效果**：滚动到技能区域时，进度条从0填充到指定百分比
- **设计**：渐变色彩，平滑过渡

### 5. 数字计数动画
- **位置**：Hero Section 统计数据
- **效果**：数字从0动态计数到目标值
- **数据**：SCI论文30篇、发明专利4项、一作论文6篇

### 6. 响应式设计
- **断点**：
  - 移动端（< 768px）
  - 平板（≥ 768px）
  - 桌面（≥ 1024px）
- **布局**：网格系统自动调整列数

### 字体系统
- **标题**：Inter 700（粗体）
- **正文**：Inter 400/500（常规/中等）
- **代码**：Fira Code（等宽）
- **层级**：
  - H1: 42px（Hero标题）
  - H2: 40px（章节标题）
  - H3: 24px（卡片标题）
  - 正文: 16px
  - 小字: 14px

- **悬停阴影**：0 20px 40px rgba(0, 0, 0, 0.4)

## 🚀 使用方法

### 1. 替换现有页面
```bash
# 备份原页面
cp about.html about.html.backup

# 重命名新页面
mv about-new.html about.html
```

### 2. 更新导航链接
确保其他页面的导航栏链接指向 `#about`

### 3. 添加图片资源
- 个人头像：`img/avatar.jpg`
- 研究相关图片（可选）

### 4. 部署到GitHub
```bash
git add about.html
git commit -m "feat: 全新设计about页面，增加3D效果和动画"
git push origin main
```
