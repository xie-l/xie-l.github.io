要上传新文件，请使用Git将文件添加到相应的目录，然后推送到GitHub。

Git上传 自动部署

如何上传文件？

1

准备文件

将你要上传的文件整理到对应的文件夹（photos/ 或 documents/）

2

使用Git提交

在终端中运行: git add . && git commit -m "添加新文件"

3

推送到GitHub

运行: git push origin main，文件会自动部署到GitHub Pages

4

访问文件

上传完成后，刷新本页面即可看到新文件
